# BillSplitter


# About 
Help split the Total bill with others in your group. Shows the percentage
paid by each member along with the tip (if at a restaurant) based of what they pay


## Deployment

* Pull from repository
* npm install - install dependicies
* npm run start - run the application
## Built With

* [Node] - runtime for backend
* [Express] - server setup
* [React] - libray
* [React-Hooks] - state management
